# 网页版五笔输入法

一个纯前端实现的网页版五笔（86版）输入法，支持 GB2312 全部 6763 个汉字，并内置拼音反查功能。

## 文件结构

| 文件 | 说明 |
|------|------|
| `index.html` | 网页入口，包含输入框、候选框 UI 和虚拟键盘容器 |
| `style.css` | 样式文件，支持 PC 端和移动端自适应布局 |
| `input-method.js` | **核心逻辑**，字典加载、编码匹配、候选分页、键盘事件处理、虚拟键盘、拼音反查 |
| `dict.txt` | **五笔字典数据**，格式：`序号: 编码 候选字1 候选字2 ...` |
| `pinyin.txt` | **拼音表数据**，用于 `z` 键拼音反查五笔编码 |

## 快速开始

直接在浏览器中打开 `index.html` 即可使用（无需服务器，但建议用本地 HTTP 服务器打开以避免 fetch 跨域限制）：

```bash
# Python 3
python -m http.server 8080

# 然后浏览器访问 http://localhost:8080
```

## 功能说明

### 核心输入功能
- **默认中文输入**：页面打开后输入法已开启
- **切换输入法**：点击按钮或虚拟键盘的「中/英」切换 中文/英文
- **输入编码**：直接输入 `a-z` 字母开始编码
- **选择候选**：数字键 `1-9` 选择候选，或 `空格` 选第一个
- **翻页**：`+` / `-` 切换候选页（虚拟键盘上为 `→` / `←`）
- **四码唯一自动上屏**：输入四个码后如果唯一匹配，自动输出到输入框（可在设置中开关）
- **清除编码**：`Enter` 清空当前编码
- **删除编码**：`Backspace` 删除最后一位编码；无编码时删除输入框文字
- **点击空白处清除编码**：点击页面空白区域（输入框和候选框以外）自动清空当前编码

### 辅助功能
- **拼音反查五笔编码**：输入 `z` + 拼音即可反查对应汉字的五笔编码（如输入 `zni` 可查"你"的编码）
- **编码提示**：候选字旁显示完整五笔编码，帮助学习（可在设置中开关）
- **中文标点转换**：中文模式下，英文标点自动转换为中文标点（如 `,` → `，`，`.` → `。` 等）
- **候选框跟随光标**：候选框可智能跟随 textarea 光标位置（可在设置中开关）
- **每页候选数**：可在设置中调整（1-9）
- **点击候选上屏**：支持鼠标/触摸点击候选词直接上屏

### 移动端适配
- **自动识别触屏设备**：在触屏设备上自动启用虚拟键盘和固定底部输入面板
- **虚拟键盘**：包含字母键及中/英切换、空格、退格、回车、翻页等功能键
- **候选框横滑**：移动端候选列表支持横向滑动浏览

## 快捷键汇总

| 按键 | 功能 |
|------|------|
| `a-z` | 输入五笔编码 |
| `1-9` | 选择对应候选词 |
| `空格` | 选择第一个候选词 |
| `+` / `-` | 下一页 / 上一页 |
| `Enter` | 清除当前编码 |
| `Backspace` | 删除编码最后一位（无编码时删除输入框文字） |
| `z` + 拼音 | 拼音反查五笔编码 |

## 字典格式

### `dict.txt`
每行格式：

```
序号: 编码 候选字1 候选字2 ...
```

例如：

```
1: a 工
2: aa 式
3: aaa 工
4: aaaa 工
7: aadn 慝 葚
```

一个汉字可能对应多个编码（一级简码、二级简码、三级简码、全码）。

### `pinyin.txt`
每行格式：

```
序号: 拼音 连续汉字串
```

例如：

```
1: a 啊阿吖嗄锕腌
2: ai 爱矮挨哎唉哀皑癌蔼埃隘捱嗳嫒瑷暧砹锿霭
```

用于 `z` 键拼音反查时检索对应汉字。

## 数据来源

- `dict.txt` 包含 GB2312 全部 6763 个汉字：
  - 一级汉字（16-55区）：3755 个
  - 二级汉字（56-87区）：3008 个
- `pinyin.txt` 提供常用汉字的拼音索引，支持反查功能

## 候选匹配策略

1. **完全匹配优先**：编码与字典中某条目完全一致的字排在前面，并高亮显示
2. **前缀匹配**：编码是当前输入前缀的字也显示在候选中
3. **去重**：同一个字只出现一次，优先保留完全匹配的版本
4. **排序**：完全匹配 > 编码短 > 字母顺序

### 拼音反查模式（z开头）
- 输入 `z` 后进入反查模式，后续字母视为拼音
- 根据拼音查找对应汉字，并显示每个汉字的五笔编码
- 反查模式不支持自动上屏和自动清空
