# 网页版五笔输入法

一个纯前端实现的网页版五笔（86版）输入法，支持 GB2312 全部 6763 个汉字。

## 文件结构

| 文件 | 说明 |
|------|------|
| `index.html` | 网页入口，包含输入框和候选框 UI |
| `style.css` | 样式文件 |
| `input-method.js` | **核心逻辑**，字典加载、编码匹配、候选分页、键盘事件处理 |
| `dict.txt` | **五笔字典数据**，格式：`序号: 编码 候选字1 候选字2 ...` |

## 快速开始

直接在浏览器中打开 `index.html` 即可使用（无需服务器，但建议用本地 HTTP 服务器打开以避免 fetch 跨域限制）：

```bash
# Python 3
python -m http.server 8080

# 然后浏览器访问 http://localhost:8080
```

## 功能说明

- **默认中文输入**：页面打开后输入法已开启
- **切换输入法**：点击按钮切换 中文/英文
- **输入编码**：直接输入 `a-z` 字母开始编码
- **选择候选**：数字键 `1-9` 或 `空格` 选第一个
- **翻页**：`+` / `-` 切换候选页
- **四码唯一自动上屏**：输入四个码后如果唯一匹配，自动输出到输入框（可在设置中开关）
- **清除编码**：`Enter` 清空当前编码
- **删除编码**：`Backspace` 删除最后一位编码
- **每页候选数**：可在设置中调整（1-9）
- **候选框跟随光标**：可在设置中开关
- **中英文标点**：中文模式下自动输出中文标点

## 字典格式

`dict.txt` 每行格式：

```
序号: 编码 候选字1 候选字2 ...
```

例如：

```
1: a 工
2: aa 式
3: aaa 工
4: aaaa 工
7: aadn 慝 葚
```

一个汉字可能对应多个编码（一级简码、二级简码、三级简码、全码）。

## 数据来源

`dict.txt` 包含 GB2312 全部 6763 个汉字：

- 一级汉字（16-55区）：3755 个
- 二级汉字（56-87区）：3008 个

## 候选匹配策略

1. **完全匹配优先**：编码与字典中某条目完全一致的字排在前面
2. **前缀匹配**：编码是当前输入前缀的字也显示在候选中
3. **去重**：同一个字只出现一次，优先保留完全匹配的版本
4. **排序**：完全匹配 > 编码短 > 字母顺序
